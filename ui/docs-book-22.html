Scope and spec-quality attachments alarm; void only on binary paths (ch. 16).

Portfolio exposure — stock read beside flow; incomplete context → amber strip, not fake confidence.

Alarm is not slash

Only on-chain facts gate pay, slash, or verdict. Cards route to courts that exist: Gate A arbiter, Gate C claim, integrity review opener.

Thin inputs → surface warnings before investor headlines.

Health is synthesis, not storage

Health — maturity, network flow, signals, exposure — assembled off-chain; stored nowhere on-chain. Never confirms, mints, or slashes.
