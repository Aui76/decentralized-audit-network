What verify promises — and what it does not

Does: self-test encoding; fetch live bytecode; re-derive PASS/FAIL `resultRoot`s; compare to chain.

Does not: full EVM replay in script — Level B execution named explicitly when it ships. Level A recomputation is falsifiable today.
