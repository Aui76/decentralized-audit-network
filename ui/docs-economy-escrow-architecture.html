Integrity ring-fence

Default integrityEscrowShareBps = 800 — 8% of each escrow deposit → integrity bucket; remainder → general escrow.

toIntegrity = toEscrow × integrityEscrowShareBps / 10_000
toGeneral   = toEscrow − toIntegrity
