Collusion and staged exploits

Discoverer pay from reserves — bounded by cap/floor (ch. 14). Staging does not inflate supply. Machine-checkable specs collapse timing games — honest hunters race reproducible proof.

If code truly conforms to filing, fight belongs at the door (Gate A) — not fake exploit.

Submit pins codehash. Identical bytecode cannot open second ordinary line. Fix → new deploy → new hash → new row. Navigation links ≠ inherited pass.

What defenses do not promise

No design eliminates every attack. Mechanism raises cost, removes timing edges, requires tool proof. Honest scope (ch. 18) is the largest collusion shrink.
