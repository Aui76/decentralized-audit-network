One loop spine; two outcomes.

Open participation, fixed rules

Researchers register into a rotating pool. No hiring desk — the gate is mechanical: procedure + proof.

RuleMeaning
Role conflictsCannot audit own bounty; claimant ≠ original auditor
HoldScales with queue position — sybil resistance
First registrarZero tokens before any confirm existed — self-bootstrap once

States strangers can read

Public states: submitted → assigned → in audit → awaiting window → confirmed. Each waiting state has permissionless liveness advance if someone goes silent.

Claimed → challenge procedure (break path) on same pin. Overlays void pre-mint — still binary tool output.
