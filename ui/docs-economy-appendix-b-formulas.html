Appendix B — Formula reference

One page** of settlement math — copy/paste for integrators.

Defaults: Appendix A. Fence post: Value vs signal.

Positive block reward

activityMint = emaSlow × emaToMintBps / 10_000
lpCap        = mintLpCapBps × lpBalance / 10_000          (skipped when lpBalance = 0)
reward       = min(activityMint, lpCap)                   when LP > 0
             = activityMint                                 when LP = 0

Manipulation discount (when emaFast / emaSlow > manipulationThresholdBps):

reward = reward × manipulationMintScaleBps / 10_000

Preview vs actual: nextPositiveBlockReward() = live estimate; auditBlockRewardMinted(id) = what minted on confirm.

Treasury on confirm

auditorMint   = reward
treasuryMint  = reward × treasuryShareBps / 10_000       (default 1× reward)
founderMint   = reward × founderShareBps / 10_000          (capped by founder bucket)

Treasury deposit split (recordDeposit):

toLP        = treasuryMint × LP_BPS / 10_000
toEscrow    = treasuryMint × ESCROW_BPS / 10_000
toIntegrity = toEscrow × integrityEscrowShareBps / 10_000
toGeneral   = toEscrow − toIntegrity

LP immediate credit capped at LP_CAP_BPS × supply / 10_000.

Discoverer pay

paid = min( bounty × boost,
            max( escrowBalance × discoveryCapBps / 10_000,
                 bounty × discoveryFloorBps / 10_000 ) )
  • General escrow only. Never mint.
  • boost from reputation — not assurance tier badges.

CAN mint

At canonicalThreshold uses at confirm:

canMint = nextPositiveBlockReward() / currentBlockSize    (BS = 1)

CAN extends application hash chain on confirm; UPG does not.

UPG mint

On structural adoption:

upgMint = upgradeAdoptMintBps × blockRewardFloor / 10_000

Module events only — not POS hash chain.

Depression floor (sketch)

When emaFast / emaSlow < depressionThresholdBps:

supplement = f(emaSlow, floorFractionBps, depressionIntensityBps)   via payFloorSupplement

Capped by maxEscrowDrawdownPerAudit. May escalate to green-light — Issuance network states · What is not live yet.

Escrow equilibrium (design)

long-run escrow outflow ≈ 16–17% of normalMint per audit
escrow inflow per confirm = 24.9% of treasury deposit
